# french-mock-tef-writing

A static site that mimics the look and feel of the TEF Canada writing test interface at
https://www.lefrancaisdesaffaires.fr/candidat/test-evaluation-francais/tef-canada/passation/,
for practicing the expression écrite section (timer, word counter, on-screen accented keyboard).

The other goal of this project was to vibe code the whole thing using Claude Code.

## Usage

No build step or server needed — just open `index.html` in your browser.
